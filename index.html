PyPandora by amoffat

Cross-platform Pandora Radio player in a single python script

Screenshot

Why?

My coworkers and I have a single music server that plays Pandora Radio in the office. We wanted the ability to control it from our desks without getting up :)

Dependencies

Web browser, Python > 2.5

Code Features

  • Ad-hoc Pandora API
  • Mini single-threaded non-blocking webserver from scratch
    • long polling (comet)
    • real-time audio streaming
  • ID3 tagging
  • Self-modifying code

Author

Extending the UI

Most people will probably want to tweak the html interface. To do that, you should know a few things: the html is embedded in pypandora.py, in the variable name "html_page". It's zlib.compressed and base64.b64encoded to save space. Fortunately, getting it out is easy (that's what she said). Just run: 

$ python pypandora.py --export
to export the html to index.html locally. You can then use this file as a base to make your changes. When you run pypandora, if it sees an index.html in its own directory, it will use that instead of the html_page variable. If you want to re-embed your index.html page back into pypandora.py, just run: 
$ python pypandora.py --import
and pypandora will slurp up index.html, zlib.compress and base64 encode it, and embed it in pypandora.py

Re-distributing (IMPORTANT)

If you choose to re-distribute pypandora, PLEASE run the following command before doing so: 

$ python pypandora.py --clean
This will clean out your saved settings from pypandora (like your login info!) You really don't want to be sharing that stuff with everyone.
